  • 단일 요청: {"jsonrpc": "2.0", "method": "tool_name", "params": {...}, "id": "uuid"}
  • 배치 요청: 여러 요청을 배열로 묶어서 처리
  • 알림: id 필드가 없는 요청 (응답 없음)
  • 스트리밍: 실시간 데이터 전송
